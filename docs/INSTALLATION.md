# Installation

Studio OS устанавливается один раз и может использоваться в любом количестве проектов.

## Структура

Studio OS (глобально)

~/.studio-os/

или

.claude/

.codex/

.mimo/

(зависит от AI)

Проект

my-project/

.studio/

docs/

src/

...

В проект копируются только:

- .studio/
- docs/

Все остальные файлы принадлежат установленной Studio OS.

## Как запустить

После установки достаточно написать:

/studio:start

Studio OS автоматически загрузит свои правила и начнёт работу с проектом.
