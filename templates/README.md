# Studio OS Artifact Templates

Templates are output assets. Load only the template required by the active Runtime.

Template section names describe canonical semantics. Translate headings and content to Project Language when creating an artifact.

All local file paths written through a template must be relative to the Target Workspace and resolve inside it. Stable web URLs may remain external. Never copy a machine-specific absolute, home, Downloads, temporary, or attachment path into an artifact.
