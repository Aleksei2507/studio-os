# Studio OS Evolution Workflow

## Goal

Create evidence-based proposals for improving Studio OS from explicit project retrospectives.

## Select When

- The user explicitly requests Studio OS Evolution.
- Project paths are provided or requested by the Evolution Runtime.

## Sequence

```text
Evolution
```

This workflow improves Studio OS itself. It is not product development and must not be used for ordinary feature growth.

## Completion

Create proposals and wait for explicit approval before changing Studio OS Runtime files.
