# Research Work Item Workflow

## Goal

Answer a bounded external or product question without starting implementation.

## Select When

- `.studio/` exists.
- The requested outcome is evidence or a decision, not a code change.

## Sequence

```text
Work Item Intake
-> Research
```

Research must record sources, findings, uncertainty, and decision impact.

Do not route to Development unless the user accepts a follow-up implementation Work Item.
